# Explore: AI Across Disciplines – Mini Case Spots

**Course:** AI Essentials
**Module:** 1
**Type:** Non-graded exploration activity

## Overview
This is a practice/explore activity built for a 100-level Introduction to AI course at a community college. 

A read-and-reflect activity presenting four short, discipline-neutral scenarios that illustrate how AI is used across different industries. Each scenario is tagged with its primary AI workload type (NLP + Classification, Predictive Analytics, Optimization/Reinforcement Learning, Conversational AI). Students read the cases, then reveal sample answers to three reflection questions to compare their thinking.

## File
`index.html` — single self-contained HTML file, no dependencies or external libraries required.

## Instructor Notes
- No student account or login is required to use this activity.
- The reveal-answer buttons use inline JavaScript; no internet connection is needed for them to function.
- Reflection questions are for student self-assessment only. No submission is expected.
